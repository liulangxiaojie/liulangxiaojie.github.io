AngularJS事件

ng-click

定义了指令定义了 AngularJS 点击事件。
开关灯思想通过这个小例子：
核心思想是通过改变flag的值来操作元素的显示和隐藏
ng-show / ng-hide 第一次初始化时，消耗性能较高; ng-if 当频繁切换时，消耗性能比较高

<!DOCTYPE html>
<html ng-app="module">//确定了angular的作用范围，后边的module是模块的名字
<head>
    <meta charset="UTF-8">
    <title>开关灯</title>
</head>
<body>
<div ng-controller="myCtrl">//控制器的作用范围，后边写的是控制器的名字
    <div ng-click="show()">{{flag?'打开':'隐藏'}}</div>
    //ng-click后边的一定是一个方法，()绝对不能少
    <div ng-hide="flag">内容</div>
</div>
</body>
<script src="angular.js"></script>
<script>
    var app=angular.module('module',[]);
    //创建一个模块
    app.controller('myCtrl', function ($scope) {
        //创建一个控制器，第一个是控制器的名字
        $scope.flag=true;//先在$scope上定义flag的值
        $scope.show= function () {
            $scope.flag=!$scope.flag;
            //每次点击都会改变flag的值
        }
    })
</script>
</html>
  • ng-hide控制DOM元素的隐藏和显示，值为true，隐藏；值为false，显示。操作css样式，不会产生作用域
  • ng-show控制DOM元素的显示和隐藏，值为true，显示；值为false，隐藏。操作css样式，不会产生作用域
  • ng-if控制DOM元素的存在与不存在，当值为true的时候，DOM元素存在，false的时候DOM元素不存在。频繁在操作DOM元素，会产生作用域，不建议使用
