数组常用方法

1.push

  • 向数组的末尾添加一个元素，改变原先数组的长度

2.unshift

  • 向数组的开头添加一个元素，改变原先数组的长度

3.shift

  • 数组的第一个元素从其中删除，并返回第一个元素的值

4.pop

  • 数组的最后一个元素从其中删除，并返回第一个元素的值

5.splice

  • 从索引n开始，删除m个； 返回值：被删除的数据以一个新数组的形式返回；原数组发生改变；
  • 从索引n开始，删除0个；添加x的新内容,添加到索引n前边; 返回值：空数组； 原数组发生改变
  • 从索引n开始(包括n)，删除m个，替换成x新内容；返回值：被删除的数据以一个新数组的形式返回；原数组发生改变；

6.slice

  • 查找索引n到m，但是不包含m,不改变原数组

7.concat

  • 数组拼接

8.sort ( function ( a,b ) {return a-b } )数组排序

  • return a-b为升序，return b-a为降序;原数组发生改变

9.reverse

  • 数组翻转,原数组发生改变

10.toString

  • 把数组转成字符串； 返回值：以逗号分隔的字符串； 原数组没发生改变；

11.join ( ‘字符’)

  • 把数组转成任意形式拼接的字符串； 返回值：任意形式拼接的字符串； 原数组没有发生改变；

12.indexOf ( )

  • 查找对应的内容，没找到对应的内容，返回-1； 找到，返回对应内容的索引；

13.find

  • 符合条件时，返回值是找到的那一项
  • 不符合条件时，返回值是undefined.

14.filter

  • filter遍历数组中的每一项，把符合条件的挑选出来，作为一个新数组返回，不会改边原数组。

15.forEach

  • 遍历数组中的每一项，里面是个匿名函数；这个匿名函数中有两个参数：item（数组中的每一项），index（数组中每一项的索引）；他没有返回值；

16.map

  • 功能和forEach()一样，但是他有返回值；
